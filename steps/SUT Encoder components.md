# Sparse Universal Transformer Encoder Components

- `_computeAttentionRepresentation`: this method computes the `attention` part of the transformer encoder it also `normalizes the input` and computes the `residual connection` between the input of the `transformer encoder` and the output of the [[Attention components]]
- `_computeFeedForwardRepresentation`: this method computes the `feedForward` part of the transformer encoder using [[Mixture of experts components]] it also `normalizes the input` and computes a `residual connection` between the output of the [[Attention components]] representation generated by the two layers in the `feedForward` module
- `_forward`: method that is contains all steps above that computes the full forward pass of the `TransformerEncoderLayerBase` mechanism
